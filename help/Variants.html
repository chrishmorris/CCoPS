Viewing a Structure with Clinical Variants

PLan uses the PV structure viewer by Marco Biasini to display PDB structures. If the structure includes human protein, PLan fetches information from ClinVar about the clinical consequences of mutations.

Getting a Structure

Go to the Structure Viewer and type the accession number of the PDB entry you are interested in, then press enter.

PLan will get the structure and display it. A table on the page lists the chains in the structure, and shows how each corresponds to the reference sequence in Uniprot.

Variants

For each chain in the structure, PLan finds the name of the gene, and looks it up in ClinVar. Getting and processing this data can take up to a minute. An aside on the page reports PLan's progress.

Some of the mutations found are single amino acid substitutions. These are all listed in a table. Each residue in the structure is coloured red if substitutions at that position are known to be pathogenic, green if known to be benign. It will be coloured yellow if there are known substitutions of unknown effect, or if both pathogenic and benign substitutions are found. Substitutions cannot be displayed if that residue was truncated from the chain.

The following variations are ignored: copy number variations, deletions, frame shifts, and synonymous codon substitutions. If a variation cannot be classified as one of these, but PLan cannot recognise it as a single amino acid subsitution, then PLan cannot display it. Such variations are ignored completely if marked as benign or of unknown significance. Pathogenic ones are added to the table. To check for these, click the arrow to expand this table.

Not for clinical use

This feature should not be used to inform medical decisions.
