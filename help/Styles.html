PLan User Interaction Style

The key values for PLan's user interface are:

  • Keeping out of the way
  • Few required fields, data entry is optional
  • Intelligent defaults
  • Plain: users' attention will be on the science not the interaction
  • Pleasant; not ugly
  • No learning curve; can be used without tuition
  • Shortcuts for expert users
  • Answers are on the screen before the question is asked
  • Modifiable; reskinning should not need a lot of resource

Logo

The logo is available in several forms, including favicons. On most pages, it should appear only in the footer. If you want to change it, edit the SVG. Unfortunately no browsers render the SVG correctly, so use the PNGs in web pages.

Other styles

The font for PLan is Vollkorn by Friedrich Althausen, available under the OFL. It is used for content, metadata, and page framework.

All boxes have slightly rounded corners at top left and bottom right, but square corners at bottom left and top right.

The background of the page is #ccc. Other shades of grey can be used for decorative purposes. Do not assume that all users can see the difference between PLan Red and grey.

All pages must flow appropriately for smaller and larger screens.

About Protein Lab Notebook (PLan)

PLan is an electronic laboratory notebook for recombinant protein expression. It will be embedded as thoroughly as possible in working life, blurring the difference between “using PLan” and “getting on with the job”. It will use the same database as PiMS, but provide a different user experience, with a smooth on ramp.
